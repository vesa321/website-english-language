Conditional Sentences in English

Understand how to express real and unreal situations!

Zero Conditional

Form: If + Present Simple, Present Simple
Use: Facts, general truths.
Example: If you heat water, it boils.

First Conditional

Form: If + Present Simple, Will + Base Verb
Use: Real future possibilities.
Example: If it rains, we will stay home.

Second Conditional

Form: If + Past Simple, Would + Base Verb
Use: Unreal or unlikely present/future situations.
Example: If I won the lottery, I would travel the world.

Third Conditional

Form: If + Past Perfect, Would Have + Past Participle
Use: Unreal past situations (regrets).
Example: If she had studied harder, she would have passed the exam.

Mixed Conditional

Form: Different tenses in 'if' and main clause
Use: Linking past condition to present result.
Example: If I had taken that job, I would be rich now.
